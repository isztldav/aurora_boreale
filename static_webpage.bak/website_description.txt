# AuroraBoreale Website Description

## Overview
A modern, responsive landing page for AuroraBoreale - an open-source ML training platform. The website uses a sophisticated design with gradient color schemes, smooth animations, and a clean professional layout optimized for showcasing machine learning technology.

## Technical Stack & Dependencies
- **Framework**: Static HTML with Tailwind CSS via CDN
- **Fonts**: Inter and Poppins from Google Fonts
- **Icons**: Inline SVG icons throughout
- **Animations**: GSAP (GreenSock) with ScrollTrigger plugin
- **Styling**: Custom CSS with Tailwind utility classes
- **JavaScript**: Vanilla JavaScript for interactions

## Color Scheme & Design System
- **Primary Brand Colors**:
  - Aurora green palette (50-900 shades): #ecfdf5 to #064e3b
  - Deep blue palette (50-900 shades): #f0f9ff to #164e63
- **Gradients**: Linear gradients combining aurora green, blue, and purple
- **Typography**:
  - Sans-serif: Inter font family
  - Display: Poppins font family
- **Effects**: Glass morphism, backdrop blur, hover transforms, glow effects

## Interactive Features & Animations

### Theme System
- **Toggle**: Sun/moon icons that swap on click
- **Storage**: Persists preference in localStorage
- **Auto-detect**: Respects system preference as default
- **Classes**: Uses 'dark' class on html element

### Mobile Responsiveness
- **Breakpoints**: Standard Tailwind responsive classes (sm, md, lg)
- **Navigation**: Collapsible hamburger menu for mobile
- **Layout**: Stacked layouts on mobile, side-by-side on desktop
- **Typography**: Responsive text sizing (text-4xl md:text-6xl lg:text-7xl)

### Animation System
- **GSAP Animations**:
  - Hero elements fade/slide in on load
  - Feature cards animate on scroll
  - Screenshot parallax effects
  - Staggered animations with delays
- **CSS Animations**:
  - Floating geometric shapes (6s infinite)
  - Glow effects on buttons
  - Hover transforms and shadows
  - Loading animations for images
- **Scroll Behaviors**:
  - Navbar hide/show on scroll direction
  - Smooth scrolling for anchor links
  - Intersection Observer for element visibility
  - Progressive image loading

### Interaction Details
- **Hover States**: All buttons and cards have hover effects
- **Focus States**: Keyboard navigation support
- **Loading States**: Images fade in when loaded
- **Error Handling**: Graceful fallbacks for missing images

- **Loading Strategy**: Lazy loading for images
- **Animation Performance**: GSAP for smooth 60fps animations
- **Bundle Size**: CDN-based dependencies reduce build complexity
- **Caching**: External CDN resources leverage browser caching
- **Responsive Images**: Proper sizing and aspect ratios

This website serves as a professional showcase for the AuroraBoreale ML training platform, emphasizing its open-source nature, technical sophistication, and ease of use through clean design and strategic content presentation.
